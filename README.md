# **생기**
##생일 하나는 기막히게 챙겨드립니다.
누구에게나 일 년에 단 하루 밖에 없는 생일은 소중한 날입니다.
축하하고 축하받는 모두가 행복할 수 있는 날을 기대하는 여러분을 위한 애플리케이션입니다.
소중한 사람과의 추억을 생기와 함께 간직하길 바랍니다.

---
# 요구사항
Android SDK 버전 24 이상
